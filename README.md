# Dialekta Handwerk Workspace

Originalgetreue Next.js-Huelle fuer die bereitgestellte HTML-Vorlage.

## Start

```bash
npm install
npm run dev
```

Die bereitgestellte HTML-Vorlage wird im Browser verlustfrei aus komprimierten Daten wiederhergestellt, damit das Design und die Funktionen unveraendert bleiben.
